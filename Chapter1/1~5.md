# 인적 자원 관리
- 관리자로 실수하는 부분이 사람을 조립식 부품처럼 관리하고자 함
    - 소프트웨어 모듈에서 내부 특성을 무시한체 인터페이스로 소통하는게 습관화 됨
- 위와 같은 방법은 인적 자원을 관리하는데 불행히도 통하지 않음
- 사람을 생각하고 관리하는 다른 방식을 살펴보기로 하자

# 1. 지금 어디선가 프로젝트가 실패하고 있다
- 기술 혁신이 필요 없는 뻔한 프로그램도 가끔씩 실패로 치닫음 -> 실패 원인은 절대 기술은 아님
##### 본질
- 실패 프로젝트에서 가장 자주 언급되는 원인으로 '정치' -> '정치'라는 단어가 다소 두리뭉실함
    - 의사소통 문제, 직원 채용 문제, 상사나 고객에 대한 환멸 등을 통틀어 '정치'라고 표현함
- 업무에서 사람과 관련 측면에서 명확한 단어 존재 -> 프로젝트의 사회학
    - 진짜 정치적 문제는 사회학의 아주 작은 병적인 일면
    - 문제의 본질이 정치가 아닌 사회학이라고 인식하면 문제를 다루기가 훨씬 수월해짐
        - ``업무에서 주요 문제는 본질적으로 기술적인 문제가 아니라 사회학적인 문제다.``
- 대다수 관리자들은 사람 걱정을 많이하지만 기술이 주요 걱정거리인 양 관리함
- 왜 사람 문제를 고민하기 보단 기술적 문제에만 매달릴까?
##### 첨단 기술 환상
- 기술적 문제에만 매달리는 이유는 `첨단 기술 환상`에 있음
    - 자신은 근본적으로 첨단 기술 산업에 종사한다고 확신하는 현상
    - 실제 첨단 기술 산업에 종사하는 사람들이 만들어 낸 `성과를 이용하는 사람`에 불과함
- `우리는 사실상 인간 의사소통 업계에 종사함`
    - 우리의 성공은 좋은 인간 관계에서 이뤄지며, 실패는 부실한 인간관계에서 얻어짐
- 관리자가 인간적인 측면보다 기술적인 측면에 집중하는 주요 원인 -> 기술적인 측면이 집중하기 더 쉽기 때문
    - 기술에 집중하는 관리자는 어두운 골목에서 열쇠를 잃어버리고 옆 골목에서 그 열쇠를 찾으려는 광대와 같음

# 치즈버거를 만들라, 치즈버거를 팔라
- 개발은 본질적으로 생산과 다름
  - 효율적인 생산을 하기 위해 다음과 같은 조치는 합리적임
    - 오작동을 없애라
    - 매장에서 빈둥거리는 직원을 엄중히 다루라
    - 직원을 교체 가능한 기계 부품으로 취급하라
    - 현재 상태를 유지하라
    - 절차를 표준화하라. 모든 일은 매뉴얼대로 하라.
    - 실험을 하지 말라.
  - 개발 환경에서 위와 같은 방식은 치명적임 -> 위와 반대 방식을 취해야함
    - 팀 사기를 꺾이고, 다른 문제에 관심을 돌리게함
##### 실수를 허용하라
- 실수를 용납하지 않은 분위기면 방어적으로 변함 -> 결과가 나쁠지도 모르는 일을 시도도 안함
- 실수를 장려하는 접근법
  - 팀원이 기회를 날려버려도 축하해줘야 마땅
##### 관리: 멍청한 정의
- '관리는 명령이다' -> 개발 환경에서 먹히지 않음
- 다그치면 당장은 생산성이 오를지 모르지만 장기적으로는 효과가 없음
  - 보충 받아야한다는 느낌은 사기를 떨어뜨림
##### 사람 가게
- 직원의 개성이 프로젝트라는 화학 반응에 활력과 효율을 불어 넣는 요소
  - 개성은 장려하고 키워줘야함
##### 현상 유지 상태의 프로젝트는 사망이다
- 개발 프로젝트의 목적은 끝내는 것임
  - 프로젝트에서 유일한 현상유지는 사후 경직 상태
- 프로젝트 관리자는 역동적으로 추진하는 데 전력으로 집중해야함
- 팀의 결속력을 높여주는 사람은 일만 열심히 하는 사람 두명의 가치를 가짐
##### 생각할 시간이 어디 있어, 일단 뛰어!
- 위험이 커질수록 좀더 신중한 접근 방식이 절실히 필요
  - 일하는 시간을 줄이고 생각하는 시간을 늘리기
- 노력이 많이 필요한 프로젝트일수록 팀원들이 더 욱 잘 소통하고 즐겨야함 -> 하나로 결속 필요
- 소프트웨어 개발자는 독서 필요


# 비엔나가 너를 기다리잖아
- '똑똑하게 일하자'라고 하지만 현실은 사생활을 희생하더라도 일하게 하는게 관리라는 사고가 만연
##### 스페인식 경영 이론
- 스페인식 경영 이론 vs 영국식 경영 이론
  - 스페인식 경영 이론: 한정된 양에 대해 효율적으로 착취 -> 신세계에서 땅과 인디언들을 착취
  - 영국식 경영 이론: 창의력과 기술 -> 산업 혁명 일으킴
- 스페인식 경영 이론은 전 세계 관리자들 사이에서 여전히 꿋꿋하게 살아나 있음
##### 집에서 받는 메시지
- 우리는 회사 밖에서는 생활 메시지들(가족, 개인사 등)을 받음
- 인생은 짧으며, 짧은 인생에서 자신이 맡은 업무보단 인생이 더 소중함
##### 초과 근무는 소용없다
- 초과 근무를 한 만큼 생활에서 해당 시간이 필요됨 -> 초과 근무가 장기적으론 효과가 없음
- 한 주를 40시간으로 쉬지도 않고 창의적으로 일하는 사람은 없음
- 유능한 직원은 초과 근무 시간만큼 적절히 쉬며 결과적으로 40시간 일함
##### 일중독자들
- 아무리 절박하게 초과 근무가 필요하더라도 사생활을 희생하면서까지 일하게 두지 말라.
  - 일 중독자들은 스페인 경영식처럼 관리하면, 일보다 인생이 중요하다는걸 깨닫는 순간 사표를 던져버림
- 초과 근무는 우수한 인력을 잃을 만큼 중요하지 않음
##### 생산성: 이기는 전투, 지는 전쟁
- 생산성을 높이고 싶다면 이직률이라는 사안을 반드시 고려해야함
- 생산성 = 이익 / 비용
  - 이익: 절감한 비용 + 업무에서 얻어진 수익
  - 비용: 업무에 투입된 팀원의 교체를 포함한 총 소요 비용
##### 한 번 더 강조하자면
- 팀원들에게 압력을 가하는 단기적인 책략은 점점 떨어지는 효율과 필연적으로 발생하는 이직률이 발생
  - 제품의 질과 일에 대한 만족감이 떨어짐
